# Security Policy

What to know which versions are currently supported for security updates?

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v0.1.0   | :white_check_mark: |

## Reporting a Vulnerability

If it is just a normal security update create an issue in the corresponding repository.
But if it is a really secret one (which should not be public) do following:
- Write an email to seesee010.kernel@gmail.com
